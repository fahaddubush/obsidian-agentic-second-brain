# 04_Knowledge

**Purpose:** Long-term semantic and evergreen knowledge.

**What belongs:** Concept notes, definitions, stable explanations, patterns, and linked lessons.

**What does not belong:** Unprocessed captures, copied sources, or temporary task state.

**Writers:** Both; AI-created claims need provenance, confidence, and review status.

**Examples:** `retrieval-augmented-generation.md`, `event-driven-architecture.md`.
