# 02_Working-Memory

**Purpose:** Short-lived context for active work and open loops.

**What belongs:** Current tasks, branch context, active bugs, hypotheses, and today's constraints.

**What does not belong:** Evergreen knowledge, immutable sources, or long-term history.

**Writers:** Both; preserve human text and make AI additions traceable.

**Examples:** `project-x - current focus.md`, `debugging checkout race.md`.
