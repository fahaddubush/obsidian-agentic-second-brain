# Agentic second-brain skill

## Purpose

Give Codex a reusable global workflow for ranked retrieval, queued evidence reconciliation, typed memory consolidation, and verification.

## What belongs here

The canonical `SKILL.md`, interface metadata, and directly referenced memory contracts.

## What does not belong here

Journal databases, personal notes, generated memories, credentials, or deployment-specific state.

## Writers

Human-governed. AI may edit the skill only under explicit authorization and must validate it before installation.
