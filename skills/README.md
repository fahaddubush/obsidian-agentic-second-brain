# Codex skills

## Purpose

Store reusable, versioned Codex skills distributed with this project.

## What belongs here

Self-contained skill folders with complete instructions, references, interface metadata, and validation.

## What does not belong here

Private runtime state, credentials, task transcripts, or machine-specific installation artifacts.

## Writers

Human-governed. AI may change a skill only when explicitly asked and must run the official skill validator.
