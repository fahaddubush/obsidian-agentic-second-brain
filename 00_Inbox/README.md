# 00_Inbox

<div class="sb-banner sb-banner--amber"><strong>Capture first.</strong> Processing and promotion happen through a reviewable workflow.</div>

**Purpose:** A low-friction capture queue for unprocessed material.

**What belongs:** Quick thoughts, pasted text, URLs, reminders, and unsorted ideas.

**What does not belong:** Curated knowledge, authoritative project state, or secrets.

**Writers:** Both. AI must label generated captures and must not rewrite human captures.

**Examples:** `2026-07-13 - project idea.md`, `links-to-review.md`.
