# 05_Episodic-Memory

**Purpose:** Dated memory of events and work over time.

**What belongs:** Sessions, decisions, milestones, incidents, and timeline entries.

**What does not belong:** Timeless concept notes or raw source files.

**Writers:** Both; AI summaries must distinguish known, inferred, and unknown details.

**Examples:** `LLM-Sessions/`, `Decisions/`, `Timeline/`.
