# LLM-Sessions

**Purpose:** Index and archive for meaningful LLM-assisted sessions.

**What belongs:** Dated, structured session summaries grouped by agent and year.

**What does not belong:** Full sensitive transcripts, invented activity, or general project documentation.

**Writers:** AI normally drafts; humans review and may correct.

**Examples:** `Codex/2026/2026-07-13 - Build vault foundation.md`.
