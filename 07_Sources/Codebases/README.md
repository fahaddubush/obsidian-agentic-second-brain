# Codebases

**Purpose:** Read-only ingestion records for external codebases.

**What belongs:** Repository identity, commit/ref, file map, inspected paths, and extraction reports.

**What does not belong:** A working checkout, secrets, dependencies, or edits to the source repository.

**Writers:** Both; AI inspection must be read-only unless separately authorized.

**Examples:** `project-x - commit abc123.md`.
