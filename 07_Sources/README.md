# 07_Sources

<div class="sb-banner sb-banner--red"><strong>Immutable evidence.</strong> Add summaries elsewhere; never rewrite captured source payloads.</div>

**Purpose:** Immutable source archive and provenance layer.

**What belongs:** Captured or referenced source material, metadata, and faithful extracts.

**What does not belong:** AI synthesis presented as source text or silent edits to captured material.

**Writers:** Humans import; AI may add separate metadata or summaries but never alter source payloads.

**Examples:** `Articles/`, `Codebases/`, `Transcripts/`.
