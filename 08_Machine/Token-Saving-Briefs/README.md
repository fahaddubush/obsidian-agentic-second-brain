# Token-Saving-Briefs

**Purpose:** The shortest useful startup context for a recurring task.

**What belongs:** Purpose, immediate state, critical constraints, next actions, and links to a current context pack.

**What does not belong:** Full history, raw transcripts, or details already available through linked notes.

**Writers:** AI writes; humans review.

**Examples:** `project-x - startup brief.md`, `second-brain maintenance.md`.
