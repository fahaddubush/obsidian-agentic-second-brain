# 08_Machine

<div class="sb-banner sb-banner--amber"><strong>Proposal space.</strong> Generated work stays traceable and reviewable before promotion.</div>

**Purpose:** Clearly separated, traceable AI-generated work awaiting use or review.

**What belongs:** Summaries, synthesis, audits, suggestions, reports, and compressed context.

**What does not belong:** Unlabeled AI prose in human notes or immutable source material.

**Writers:** AI writes; humans review, annotate, promote, or reject.

**Examples:** `Summaries/`, `Context-Packs/`, `Project-Audits/`.
