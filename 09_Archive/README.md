# 09_Archive

**Purpose:** Material no longer active but retained for history and recovery.

**What belongs:** Superseded notes, finished working memory, retired projects, and deprecated machine output.

**What does not belong:** Deleted content, current sources, or items moved without preserving links and provenance.

**Writers:** Both; archival moves require care and destructive deletion requires explicit approval.

**Examples:** `2025/retired-project/`.
