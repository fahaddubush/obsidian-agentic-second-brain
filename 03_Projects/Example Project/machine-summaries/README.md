## Purpose

Store project-specific AI summaries before human acceptance.

## Belongs

Timestamped, source-linked, clearly generated summaries.

## Does not belong

Raw code, human notes, or summaries without provenance.

## Writers

AI writes; humans review, annotate, and promote conclusions to shared project notes.
