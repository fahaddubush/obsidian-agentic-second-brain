# 03_Projects

<div class="sb-banner sb-banner--blue"><strong>Project memory, not project code.</strong> Ingestion is read-only unless a separate request authorizes changes.</div>

**Purpose:** Durable memory organized by project.

**What belongs:** Project context, goals, architecture, decisions, tasks, bugs, changelog, lessons, ingestion logs, and review subfolders.

**What does not belong:** Project source code, vendored dependencies, or unverifiable claims.

**Writers:** Both; AI writes reviewable project documentation and never changes the source project without permission.

**Examples:** `project-x/context.md`, `course-capstone/architecture.md`.
